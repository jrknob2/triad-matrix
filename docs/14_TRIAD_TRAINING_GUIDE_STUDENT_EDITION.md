# TRIAD TRAINING GUIDE — STUDENT EDITION

This guide shows how to use TRIADS (3-note sticking patterns)
to build control, flow, speed, hand-to-hand consistency,
kick independence, and musical phrasing around the kit.

You will learn:

- The Triad Matrix
- The 8 hand only triads
- The 19 kick trades
- Triad accenting
- Voice assignments
- Musical phrasing on the kit (flow)

The guide covers beginner, intermediate and advanced skill levels.

## TRIAD MATRIX (REFERENCE)

```text
      R     L     K
   +-----+-----+-----+
RR | RRR | LRR | KRR |
   +-----+-----+-----+
LL | RLL | LLL | KLL |
   +-----+------+----+
RL | RRL | LRL | KRL |
   +-----+------+----+
LR | RLR | LLR | KLR |
   +-----+-----+-----+
KK | RKK | LKK | KKK |
   +-----+------+----+
RK | RRK | LRK | KRK |
   +-----+-----+--——-+
LK | RLK | LLK | KLK |
   +-----+-----+--——-+
KR | RKR | LKR | KKR |
   +-----+-----+-----+
KL | RKL | LKL | KKL |
   +-----+-----+-----+
```

Notes: The columns and rows group all the triads together that have the indicated sticking. Hand only triads are bold.

## HAND-ONLY TRIADS

```text
A = RLL          B = LRR
      ^                ^
C = LLR          D = RRL
      ^                ^
E = RLR          F = LRL

G = RRR          H = LLL
```

The caret (^) marks the first SINGLE stroke. That stroke gets the natural accent by default. Double strokes (LL or RR) are ghosted unless we explicitly stated otherwise.

## VOICES

```text
S = Snare       T1 = Tom 1
T2 = Tom 2      FT = Floor Tom
HH = Hi-Hat     HK = Hi-Ht + Kick (*)
HR = RLR with Hi-Hat + Kick on both Rs
HL = LRL with Hi-Hat + Kick on both Ls
CK = Crash + Kick (*)
```

(*) = simultaneously

## DEFAULT PERFORMANCE RULES

1. Accents
   - Accent the SINGLE stroke in the triad.
   - Double strokes (LL or RR) are ghost notes.
   - Triads with no singles (LLL or RRR) have no built-in accent. Experiment!

2. Voices
   - Everything defaults to the snare (S).
   - A voice tag (AT1, AFT, BK, etc.) tells you where the ACCENTED single stroke is played.
   - Ghost notes stay on snare unless explicitly stated otherwise

## VOICE ASSIGNMENT EXAMPLES

1. Triad A (all on snare):

   ```text
   R L L
   S S S
   ```

   or short hand (default) just:

   ```text
   A
   ```

2. Triad A X 2 with 2nd accented on Tom 1 (repeat):

   ```text
   A->AT1
   ```

3. Key ideas:
   - Voice tags never change sticking.
   - They only move the accented SINGLE stroke.
   - Ghost notes remain on snare unless otherwise stated.

## KICK TRIADS

```text
AK = KRR          BK = KLL          CK = RKK
DK = LKK          EK = RKR          FK = LKR
GK = RKL          HK = LKL          IK = LLK
JK = RRK          KK = LRK          LK = RLK
MK = KRL          NK = KLR          OK = KRK
PK = KLK          QK = KKR          RK = KKL
SK = KKK
```

## BEGINNER EXERCISES (SNARE ONLY)

### LESSON RULES

- No toms, no kick, no voice tags.
- Accents and ghosting follow the default rules.
- Repeat for 10 minutes each
- Evenness over speed. Speed will come all by itself
- Repeat smoothly with no gaps from the end back to the beginning.

1. Basic hand triads (repeat each one)

   ```text
   A → A → A → A

   B → B → B → B

   A → A → B → B

   B → B → A → A

   A → B → A → B

   B → A → B → A
   ```

## INTERMEDIATE EXERCISES (INTRODUCING VOICES AND KICK)

### LESSON RULES

- Repeat for 10 minutes each
- Evenness over speed. Speed will come all by itself
- Repeat smoothly with no gaps from the end back to the beginning.

### Phase 1 - Alternating Between Hands and Voices

```text
A → AT1 → A → AT1

B → BT1 → B → BT1

AT1 → AT2 → B -> BHK

B → B → A → A

A → B → A → B

B → A → B → A
```

### Phase 2 - Adding Kick

```text
A → AK → A → AK
B → BK → B → BK
```
